[Created in Go](https://golang.org/doc/install), to build from source.

Clone this repo.

```
cd go/src
git clone https://github.com/Defacto2/sitemap.git
```

Install dependancies.

```
go get github.com/go-sql-driver/mysql
```

```
cd sitemap
go build
```

`./sitemap` or `sitemap`
